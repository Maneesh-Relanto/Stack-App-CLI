## Quick Start

### Using Stack-App-CLI

```bash
# Interactive mode
npx create-stack-app new my-app

# With specific template
npx create-stack-app new my-app --template nextjs-saas

# List all templates
npx create-stack-app list

# Browse by language
npx create-stack-app new my-app --language TypeScript

# Browse by category
npx create-stack-app new my-app --category backend
```


